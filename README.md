# Reimagine

Simple React image uploader.

## Installation

`npm install --save reimagine`

or

`yarn add reimagine`

## Example
https://codesandbox.io/s/hungry-mayer-5q5sl?fontsize=14

## Usage

```javascript
import Reimagine from "reimagine";

<Reimagine
  width={200}
  height={200}
  onSelectImage={uploadImage}
  onEdit={editImage}
/>;
```
